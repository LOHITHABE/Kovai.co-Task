## Overview ##
-> This project analyzes daily public transport passenger journeys by service type using exploratory data analysis (EDA), visualizations, and forecasting.

## Dataset ##
-> Daily_Public_Transport_Passenger_Journeys_by_Service_Type_20250603.csv

-> Contains daily passenger counts for services: Local Route, Light Rail, Peak Service, Rapid Route, School and other.

## Exploratory Data Analysis (EDA) ##
-> Converted Date to datetime and set it as index.

-> Checked missing values and data types.

-> Visualized passenger trends over time.

## Key Insights ##
-> Weekday demand is higher than weekends.

-> School service depends on academic schedules.

-> Peak services show higher variability.

-> Passenger demand follows weekly seasonal patterns.

-> Local Route and Light Rail have stable long-term trends.

## Forecasting ##
-> SARIMA model is used to forecast passenger demand.

-> EDA insights help identify seasonal patterns and trends for accurate predictions.
